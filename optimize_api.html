Optimizaiton API¶

The following plans form the nabs support for generic optimization problems.

Optimization Plans¶

maximize Maximize the value of an ophyd.Signal
minimize Minimize the value of an ophyd.Signal
walk_to_target Walk the motor until a signal reaches our target
optimize Generic optimization method

Optimization Methods¶

golden_section_search Use golden-section search to find the extrema of a signal
